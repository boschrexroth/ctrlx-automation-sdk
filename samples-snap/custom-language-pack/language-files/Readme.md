# language-files directory

Place the translated language files into a subfolder prefixed with the language code, e.g. fr for french.

The files in the folder should be structured like this:

```
./language-files
  fr/
  └ webapp/
    └ *.fr.json 
  └ *.diagnostics.fr-FR.json
  └ *.package-manifest.fr.json
```
