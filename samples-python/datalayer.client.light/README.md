# README datalayer.client.light

## Introduction

The sample demonstrates how to read values from __ctrlX Data Layer__.

## Function Description

A subscription is used to get values by data change event in a deterministic publish interval. The single read is performed in an endless loop.

## Implementation Description

The module __main.py__ connects to the __ctrlX Data Layer__ and handles the __ctrlX Data Layer__ access.
